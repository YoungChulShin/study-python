# 제어문
## if 문
정의
```py
if 조건:
    조건의 내용
elif 조건:
    조건의 내용
else:
    조건의 내용
```

조건부 표현식
```py
# score가 60이상일 경우 "success" 아니면 "failure"를 출력하는 내용

# 조건문
if score >= 60:
    message = "success"
else:
    message = "failure"

# 표현식
message = "success" if score >= 60 else "failure"
```

## while 문
정의
```py
while 조건:
    내용
```

특별한 문법
- break: while문 빠져나오기
- continue: while문의 처음으로 가기
